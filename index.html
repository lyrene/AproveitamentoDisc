Sistema de Aproveitamento de Estudos

Esta ferramenta auxilia na extração e organização dos dados necessários para a análise de Aproveitamento de Estudos na universidade. O sistema é dividido em três módulos independentes, que podem ser usados juntos ou separadamente.

Módulo A — Importar Processos (arquivo HTML do SIPAC)

Entrada esperada: um arquivo .html contendo o resultado da consulta de processos no SIPAC (lista com vários processos ou a página detalhada de um único processo).

O sistema irá extrair automaticamente:

  • Número do processo
  • Nome do aluno
  • Origem do processo
  • Destino do processo
  • Situação do processo
Acessar Módulo A

Módulo B — Processar Requerimentos (PDFs dos alunos)

Entrada esperada: um conjunto de arquivos .pdf contendo os Requerimentos de Aproveitamento de Estudos emitidos pelo SIGAA.

Cada PDF deve conter obrigatoriamente:

  • Nome do aluno + matrícula
  • Disciplinas de Origem (curso(s) anterior(es))
  • Disciplinas de Destino (solicitadas no requerimento)

O sistema irá realizar automaticamente:

  • Associação aluno ↔ processo usando o nome
  • Extração das disciplinas de origem
  • Extração das disciplinas de destino
  • Criação do CSV intermediário de aproveitamentos

Observação importante:
O coordenador poderá revisar e modificar as disciplinas de destino e suas correspondências após baixar o CSV.

Acessar Módulo B

Módulo C — Gerar Fichas de Aproveitamento

Entrada esperada: o arquivo CSV revisado pelo coordenador, contendo uma ou mais linhas para cada disciplina que o aluno deseja aproveitar.

O sistema irá gerar automaticamente:

  • Ficha de Aproveitamento de Estudos (PDF) por processo
  • Arquivo ZIP contendo todas as fichas geradas
Acessar Módulo C
